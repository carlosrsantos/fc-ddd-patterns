# Desafio Full Cycle - DDD: Modelagem Tática e Patterns

Implementação de modelagem tática com DDD, padrões de projeto e testes unitários utilizando Typescript.

## Tabela de Conteúdos

- [Requisitos](#requisitos)
- [Instalação](#instalação)


## Requisitos

- Node.js
- TypeScript
- Jest

## Instalação

1. **Clone o repositório**:

   ```bash
   git clone https://github.com/carlosrsantos/fc-ddd-patterns.git

   cd <app-folder>
   ```

2. **Instalar dependências**:
  
  ```bash
   npm install
   ```

3. **Executar testes unitários**:

  ```bash
  npm run test
   ```
